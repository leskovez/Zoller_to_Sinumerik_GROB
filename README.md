# Zoller_to_Sinumerik_GROB

This program is designed to parse the zidcode string given when scanning a zidcode tag generated by the zoller tool measuring system. It then formats the radius and length value into a text file to be read by a macro on a GROB with a siemens sinumerik 840d sl controller but I'm sure it can be easily adapted for other controllers and / or macros for passing the data into the tool offsets. The UI and code are definitely a little clunky but it does work, you have been warned lol.
